电脑设置总结

  1. 电脑命令总结
    任务计划 taskschd.msc
    组件服务 dcomcnfg
    设备管理器 devmgmt.msc
    服务 services.msc
    系统配置 msconfig
    计算机管理 compmgmt.msc
    程序管理 appwiz.cpl

  2. win10打开ie自动跳转edge浏览器
    win+Q键搜索internet选项，选中高级，把勾选“启用第三方浏览器扩展”去掉就不会自动跳转

  3. window设置开机自动登陆

    1. win+R输入regedit进入注册表，HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\PasswordLess\Device，找到下面的DevicePasswordLessBuildVersion修改值为0
    2. win+R输入netplwiz确定，去掉红色框中的对钩，点击应用会弹出窗口让输入开机密码，输入后确定，然后重启电脑就可以。

前端配置操作

npm

  1. npm项目初始化操作

npm init 项目初始化后会生成package.json文件，类似于java中的pom.xml文件

#按照提示输入相关信息，如果是用默认值则直接回车即可。

#name: 项目名称

#version: 项目版本号

#description: 项目描述

#keywords: {Array}关键词，便于用户搜索到我们的项目

npm init -y 直接初始化项目，不用一步一步确定

  1. npm下载js依赖
    1. npm install 依赖名称 例如：（npm install jquery）
    2. npm install 指定依赖版本 （npm install jquery@2.1.x）

下载npm依赖之前最好修改下npm镜像地址
npm官方的管理包都是从http://npmjs.com下载，这个网站国内访问速度慢，推荐使用淘宝的npm镜像http://npm.taobao.org

npm install 命令使用

  1. npm 全局安装
    全局安装的目录在 用户目录\AppData\Roaming\npm\node_modules
    npm intall -g webpack
