Oracle 创建用户并授权

创建表空间

进入dos界面，输入 sqlplus / as sysdba;以dba的权限登陆oracle (sqlplus test/test as sysdba);test指用户名，/后的test是密码。

说明：

testSpace：表空间名
datafile 数据库文件及路径

创建用户，并设置默认表空间

说明：

user后的test是用户名
by后的test 是 密码
