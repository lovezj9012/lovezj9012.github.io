Linux安装python3

Tools
Tools
发布日期:   2024-08-07
文章字数:   616
阅读次数:  

环境

操作系统：CentOS7

安装依赖等

  • wget下载命令没有则安装
# 非root用户需要加上sudo
sudo yum install -y wget
  • 检查是否安装了编译套件gcc
# 非root用户需要加上sudo
sudo yum -y install gcc
  • 依赖
# 非root用户需要加上sudo
yum -y install zlib*  
yum -y install libffi-devel

下载python

下载python到/opt/soft目录下并解压

wget https://www.python.org/ftp/python/3.12.4/Python-3.12.4.tar.xz

# 后台下载
wget -b https://www.python.org/ftp/python/3.12.4/Python-3.12.4.tar.xz

# 后台递归下载文件
wget -br https://www.python.org/ftp/python/3.12.4/

# 下载文本中的链接内容
# test.txt https://wordpress.org/latest.zip https://ftp.drupal.org/files/projects/drupal-8.4.5.zip
wget -i test.txt

# 解压
tar -xvf Python-3.12.4.tar.xz

编译安装python

创建python3文件夹

sudo mkdir /usr/local/python3

配置python安装路径

# 进入解压python路径
cd /opt/soft/Python-3.12.4
# 配置路径
./configure prefix=/usr/local/python3
# 编译
make&&make install

配置python环境变量

  • /etc/profile （适用于所有用户）
  • ~/.bashrc （适用于当前用户）
  • /etc/bashrc （适用于所有用户）

配置profile文件

export PYTHON3=/usr/local/python3
export PATH=${PATH}:${PYTHON3}/bin

#保存配置后
source /etc/profile

建立软连接

ln -s /usr/local/python3/bin/python3 /usr/bin/python3

验证是否安装成功

python3 -V

编译源代码安装python3

操作系统：Ubuntu16.04

下载解压源代码

# -c 表示断点续传
wget -c https://www.python.org/ftp/python/3.12.0/Python-3.12.0.tgz

tar -xzvf Python-3.12.0.tgz

配置

# 进入Python-3.12.0文件夹
cd Python-3.12.0

# 配置 --prefix是可选参数
sudo ./configure --prefix=/usr/bin/python3.12

安装依赖包

python3必须先安装依赖，python2可以不用

sudo apt-get install build-essential gdb lcov pkg-config \
      libbz2-dev libffi-dev libgdbm-dev libgdbm-compat-dev liblzma-dev \
      libncurses5-dev libreadline6-dev libsqlite3-dev libssl-dev \
      lzma lzma-dev tk-dev uuid-dev zlib1g-dev libmpdec-dev

编译

make

安装

sudo make install

# 编译和安装可以一步执行
sudo make && make install

配置软连接

sudo ln -s /usr/bin/python3.12/python3 /usr/bin/python3.12

# 如果有问题可以加-f 表示可以覆盖
sudo ln -s -f /usr/bin/python3.12/python3 /usr/bin/python3.12

切换linux下默认python版本

# 删除python的软连接
rm /usr/bin/python
# 新建python3的软连接
ln -s /usr/bin/python3 /usr/bin/python

安装wxpython

# -i后指定国内源
pip install wxpython -i https://pypi.tuna.tsinghua.edu.cn/simple
