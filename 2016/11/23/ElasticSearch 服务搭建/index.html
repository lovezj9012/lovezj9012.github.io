ElasticSearch 服务搭建

一、搭建环境

操作系统：Win7

ElasticSearch:1.73

Java JDK:1.7

二、搭建操作

  1. 安装Java JDK,然后配置系统环境变量，新建CLASS_PATH（jre所在目录 【.;D:\Program Files\Java\jre7\lib】）和JAVA_HOME(D:\Program Files\Java\jdk1.7.0_80)

    在系统变量Path最后位置输入;%JAVA_HOME%\bin
  2. 在系统变量Path中输入elasticseach的路径（D:\elasticsearch\elasticsearch-1.7.3\bin），根据文件所在位置更改；这样可以直接执行elasticsearch安装命令（service intsall）、启动服务（service start）、停止服务（service stop）、卸载服务（service remove）
  3. 打开dos命令窗口，输入service install，看到下图表示安装服务成功，然后输入service start 启动服务
  4. 在浏览器中输入localhost:9200如下图所示表示Elasticsearch启动成功
  5. 上图中的name和cluster_name可以通过修改config文件夹下的elasticsearch.yml的配置文件来修改
  6. 安装head插件输入plugin -install mobz/elasticsearch-head即可安装head插件，具体的操作步骤也可以参考github
  7. 安装google浏览器的插件sense，其他的版本安装方法可以查看文章 elasticsearch 插件 大全 中的marvel插件

还有其他的插件安装方法都类似

elasticsearch集群监控插件bigdesk (安装命令 plugin -install lukas-vlcek/bigdesk)

集群资源查看和查询插件 kopf （安装命令 plugin -install lmenezes/elasticsearch-kopf）

调试查询 inquisitor （安装命令 plugin -install polyfractal/elasticsearch-inquisitor）

类似sql语法的查询插件 elasticsearch-sql （安装命令 plugin -install NLPchina/elasticsearch-sql）

更多插件可以参考 http://www.searchtech.pro/elasticsearch-plugins

github有一个可以定制msi安装elasticsearch工具感兴趣可以看看https://github.com/salyh/elasticsearch-msi-installer
