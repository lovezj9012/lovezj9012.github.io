hexo搭建博客

1、安装Git

2、安装nodejs

3、安装hexo,安装完成可以使用hexo -v命令查看是否安装成功
1、初始化
2、进入blogs文件夹
    
    blogs文件夹下有
        node_modules: 依赖包
        public：存放生成的页面
        scaffolds：生成文章的一些模板
        source：用来存放你的文章
        themes：主题** 
        _config.yml: 博客的配置文件**
3、运行服务,可以浏览网站了

4、GitHub创建仓库（仓库名 youname.github.io）

5、生成ssh添加到GitHub

 a、在gitbash中配置名称和邮箱
    git config --global user.name "youname"
    git config --global user.email "youremail"
 b、检查是否配置成功
    git config user.name
    git config user.email
 c、创建ssh，一直回车
    ssh-keygen -t rsa -C "youemail"
 d、进入github的setting找到ssh and gpg keys
    新建sshkey 将id_rsa.pub中的内容复制到keys中
 e、在gitbash中验证是否成功
    ssh -T git@github.com

6、将hexo部署到github

a、修改_config.yml文件
   deploy:
    type: git
    repo: https://github.com/YourgithubName/YourgithubName.github.io.git
    branch: master  根据自己的分支名修改
b、安装deploy-git
    npm install hexo-deployer-git --save
c、执行命令
   hexo clean      清除之前生成的内容
   hexo generate   生成静态内容，缩写 hexo g
   hexo deploy     部署 缩写 hexo d

7、新建分类、标签、关于

a、分类
   hexo new page categories  --在source文件夹下回新建一个
                               categories文件夹 
                               其中有一个index.md,在date之后加上type:'categories'
b、标签
    hexo new page tags      --在source文件夹下回新建一个tags文件夹
                               其中有一个index.md,在date之后加上type:'tags'
c、关于
    hexo new page about     --在source文件夹下回新建一个about文件夹
                               其中有一个index.md
